GoogleCalendarGenerator

Plugin generator for Ruby On Rails Allow to display google Calendar events in a Rails application.

  • Project page: benjamin.francisoud.googlepages.com/googlecalendar
  • RubyForge page: rubyforge.org/projects/googlecalendar
  • API reference: googlecalendar.rubyforge.org/doc

Install

  • [railsapp]>ruby script/plugin install svn://rubyforge.org//var/svn/googlecalendar/plugins/googlecalendar

Dependancy

You need to install the CalendarHelperPlugin:

  • [railsapp]>ruby script/plugin install topfunky.net/svn/plugins/calendar_helper
  • [railsapp]/vendor/plugins/calendar_helper>rake rdoc
  • [railsapp]>ruby script/generate calendar_styles

For more informations about the calendar plugin: wiki.rubyonrails.com/rails/pages/Calendar+Helper+Plugin

Usage

  • [railsapp]>ruby script/generate googlecalendar
